As of the current update (28-07-2023) the following functionality is possible to perform using this application:

  1. View and search users: Access these users in the users topnav button. You cannot add users so please use the azure portal to add users first and then search them in the portal for the current update.
  2. Modify the directory extension property values for users: You can access the user in the user topnav button and then click on any particular user. A set of directory extension properties scoped to the user will pop up which you can edit with the values.
  3. View groups: You can view groups through the view groups button in the topnav.
  4. Modify directory extension property values for groups: Similar to users.
  5. Add directory extension properties to the user object: You can do this by selecting Template user properties in the topnav. You can view all properties you have already added as well as add new properties.

Features which are broken:

  • Currently, the feature to add directory extension properties to groups is broken. The delete capability is also broken as of the latest update.
  • The feature to add groups is broken and the button will not work.
  • The feature to delete extension properties of users and groups is broken.

Expectations for the new update:

  • Fixing all bugs and broken code.

New features to be implemented:
